# covid19-data-analysis-dashboard
COVID-19 Power BI Dashboard
Project Overview
This project presents an interactive COVID-19 Dashboard built using Power BI to analyze global pandemic trends.
It provides insights into confirmed cases, deaths, recoveries, and country-wise comparisons through dynamic visualizations.

Features
1. Country-wise COVID-19 analysis
2. Trend analysis (cases, deaths, recoveries over time)
3. Interactive filters and slicers
4. Global summary metrics
5. Data visualization using charts and maps

Tools & Technologies
1. Power BI
2. Microsoft Excel / CSV (Data Source)
3. Data Cleaning & Transformation
4. Data Visualization
